# Guild — Agent Team Configuration

Guild contiene la configuración de comportamiento del equipo de agentes: skills, prompts, reglas de equipo, y configuraciones de OpenCode.

## Contenido

- `guild-agents.json` — Bundle de agentes para mergear en opencode.json
- `AGENTS.md` — Team personality y reglas de conducta
- `skills/` — Skills de SDD y workflows
- `commands/` — Comandos SDD para OpenCode
- `prompts/` — Prompts de agentes (SDD, creative, research, strategy)

## Deploy

Ejecutar como post-install de flint:
```bash
sudo ./deploy.sh
```
